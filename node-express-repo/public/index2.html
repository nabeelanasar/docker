API End Points


API Version Methods Description
Blogs
/blogs 1 GET Retreive records from blogs
/blogs/id 1 GET, PATCH, PUT, DELETE Retreive/Update/Delete records from blogs
/blogs?$select[]=author 1 GET Retreive one field from blogs
/blogs?$sort[author]=1 1 GET Sort records in acending order(-1 decending order)
/blogs?$limit=1 1 GET Limit the number of records
/blogs?$skip=1 1 GET Skip the number of records
/blogs?q=categories 1 GET Retreive all records from blogs_categories
/blogs?q=categories&$limit=10&$skip=1 1 GET Get supports both limit and skip
